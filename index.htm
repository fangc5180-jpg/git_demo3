Git 版本控制笔记

常用命令：

git reset --hard [版本号]

HEAD 表示当前版本，HEAD^ 表示上一个版本，HEAD~n 表示前n个版本

概念 说明 示例
HEAD 当前版本指针 HEAD 指向当前提交
HEAD^ 上一个版本 git reset --hard HEAD^
HEAD~3 前3个版本 git reset --hard HEAD~3
